
SISTEMA ESTOQUE WEB - Google Sheets

1️⃣ Instalar dependências:
pip install -r requirements.txt

2️⃣ Abrir a planilha Google com abas:
- usuarios
- produtos
- movimentacoes

3️⃣ Compartilhar a planilha com o email da conta de serviço (está no credencial.json)

4️⃣ Executar local:
streamlit run app.py

5️⃣ Deploy na nuvem:
- Crie conta em https://streamlit.io/
- Suba este projeto no GitHub
- Conecte seu repositório no Streamlit Cloud
- Deploy gratuito e online!
